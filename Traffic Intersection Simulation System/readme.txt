The program begins in the fail-safe mode. This mode can also be activated when 'fsmode' is entered.

The static cycle mode is activated when 'scmode' is entered.

The accelerate-time mode is activated when 'atmode num' is entered with num representing the speed ranging from 1 to 100.

Available commands are displayed when '?' Is entered.

WHAT DOES NOT WORK?
Program stops when '?' is entered.
